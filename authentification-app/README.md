# authentification-app with docker-compose

## Project setup start all contatiners
```
docker-compose up
```
## Project setup in background
```
docker-compose up -d
```

### logs project setup
```
docker-compose logs
```

### Compiles and minifies for production
```
docker-compose build
```

### app
```
http://localhost:8080/
```

### phpmyadmin
```
http://localhost:8081/
```
